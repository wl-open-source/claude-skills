# Dokumenttyp-Kürzel

Kürzel stehen für den **Dokumenttyp**, nicht für den Absender/Vertragspartner (der gehört in den Freitext-Teil des Dateinamens). Bei einem neuen Dokumenttyp: sinnvolles, kurzes Kürzel ergänzen statt ein unpassendes bestehendes zu erzwingen — Konsistenz über die Zeit zählt mehr als das perfekte Kürzel beim ersten Mal.

| Kürzel | Dokumenttyp |
|---|---|
| RG | Rechnung |
| ANG | Angebot |
| VTR | Vertrag |
| AB | Auftragsbestätigung |
| MAHN | Mahnung |
| KVA | Kostenvoranschlag |
| BEW | Bewerbungsunterlage |
| ZEUG | Zeugnis/Zertifikat |
| PROT | Protokoll |
| PRES | Präsentation |
| ANL | Anleitung/Handbuch |
| VERS | Versicherungsunterlage |
| STEU | Steuerunterlage |
| GEHALT | Gehaltsabrechnung |
| VERD | Verdienstbescheinigung |
| KUEND | Kündigung |
| ANTR | Antrag |
| BESCH | Bescheid (Behörde/Amt) |
| TICKET | Reise-/Veranstaltungsbuchung |
| EXPORT | Datenexport (CSV, JSON o.ä.) |
| NOTIZ | Notiz/Freitext-Dokument |

## Sonderfälle (eigenes Format, siehe SKILL.md Ausnahmetabelle)

- Fotos → nicht einzeln umbenennen
- Screenshots → `Screenshot` als Kürzel
- Installer/Setup-Dateien → nicht umbenennen
- Code-/Projektdateien → nicht umbenennen
